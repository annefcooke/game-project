Instructions


Test your memory and copy Simon!

Choose your game mode and click start.

Game Modes:

  • Easy: Slowest mode - copy the sequence.
  • Difficult: Increased speed! Double points.
  • Double: Sequence increases by 2 each level. Triple points!
  • Reverse: You need to put the sequence in backwards! Triple points.

There are 15 levels in each mode. See if you can complete all levels!
